# Welcome to Rust in 90 Days

**Transform yourself from a Rust newcomer to a confident systems programmer in just 90 days with 10-minute daily lessons.**

## What You'll Achieve

By the end of this journey, you'll be able to:

- ✅ **Write safe, fast systems code** with confidence
- ✅ **Understand ownership and borrowing** intuitively 
- ✅ **Build CLI tools, web servers, and data structures** from scratch
- ✅ **Debug complex compiler errors** efficiently
- ✅ **Apply async programming** for concurrent applications
- ✅ **Navigate the Rust ecosystem** and choose the right crates
- ✅ **Contribute to open source** Rust projects

## The Philosophy Behind This Course

### 🧠 Science-Based Learning
This course is built on proven learning principles:
- **Spaced repetition** - concepts are revisited and reinforced
- **Progressive complexity** - each lesson builds naturally on the last
- **Active practice** - you write code every single day
- **Immediate feedback** - the Rust compiler is your constant teacher

### ⚡ Micro-Learning Approach
**10 minutes per day** is the sweet spot for:
- **Sustainable progress** without burnout
- **Consistent habit formation** that sticks
- **Deep focus** on one concept at a time
- **Fitting into any schedule** - even the busiest professionals

### 🎯 Project-Driven Curriculum
You'll build **7 progressively complex projects**:
1. **Temperature Converter** (Day 11) - First CLI tool
2. **CLI Calculator** (Day 14) - Parsing and evaluation
3. **Text Analyzer** (Day 27) - Working with strings and files
4. **Config Parser** (Day 40) - Structured data handling
5. **JSON-like Data Structure** (Day 42) - Custom types and serialization
6. **Word Frequency Counter** (Day 55) - Collections and algorithms
7. **Generic Data Store** (Day 69) - Traits and generics in action
8. **Graph Data Structure** (Day 83) - Advanced memory management
9. **Micro Web Server** (Day 90) - Async programming capstone

## Course Structure Overview

### 📅 Daily Commitment
- **10 minutes of focused learning**
- **1 concept per lesson** with clear examples
- **Practice exercise** to reinforce the concept
- **Progress tracking** to maintain momentum

### 📚 Weekly Themes
Each week focuses on a major Rust concept area with 7 related lessons that build understanding progressively.

### 🏗️ Project Weeks
Weeks 2, 4, 6, 8, 10, 12, and 13 include hands-on projects that integrate everything you've learned.

## Success Tips

### 🎯 For Maximum Success:
1. **Commit to the daily habit** - consistency beats intensity
2. **Code along with examples** - don't just read passively
3. **Experiment beyond the exercises** - break things and fix them
4. **Read compiler error messages carefully** - they're educational
5. **Join the Rust community** on Discord/Reddit for support
6. **Don't skip the difficult topics** - ownership is crucial

### ⚠️ Common Pitfalls to Avoid:
- Rushing through ownership concepts (weeks 3-4)
- Skipping the practice exercises
- Not reading error messages thoroughly
- Trying to memorize instead of understanding patterns
- Giving up when the borrow checker fights you

## Tools You'll Need

### Required Setup:
```bash
# Install Rust (15 minutes)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Verify installation
rustc --version
cargo --version
```

### Recommended Tools:
- **VS Code** with rust-analyzer extension
- **Git** for version control practice
- **Terminal/Command Prompt** comfort

### Optional Enhancements:
- **cargo-watch** for automatic rebuilding
- **clippy** for additional linting (included with Rust)
- **rustfmt** for code formatting (included with Rust)
